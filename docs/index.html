⚠️ Web版について

本プロジェクトはAndroid専用アプリとして開発されており、以下の理由によりWeb版での完全な機能提供は困難です：

  1. 性質上コアな価値をWeb版でほとんど提供できない
    • モバイル特化のUX設計（タッチ操作、画面サイズ最適化）
    • プラットフォーム固有のライフサイクル管理
  2. 実装が間に合わない
    • ハッカソン期限内でのWeb版対応が技術的に困難
    • Flutter WebでのPlatform.isAndroid対応等、大幅な条件分岐が必要
このページでは、システム構成・技術詳細・デモ動画を通じてプロジェクトの全容をご紹介しています。

📱 プロジェクト概要

WellFinは、現代の忙しい生活の中で生産性向上と習慣形成を支援するAI駆動型アプリケーションです。Google Cloud AI技術を活用し、ユーザー一人ひとりに最適化されたパーソナライズ体験を提供します。

🎯 主要機能

  • インテリジェントタスク管理 - AI分析による優先度自動設定、スケジュール最適化
  • 習慣トラッキング - 科学的根拠に基づいた習慣形成支援、継続率向上のためのAI提案
  • 目標設定・管理 - SMART目標設定支援、達成可能性分析
  • パフォーマンス分析 - 生産性メトリクス可視化、改善点のAI提案
